# Arcane Ledger Attributions

This file is the starting ledger for sources used by Arcane Ledger. It is intentionally short and should grow as the project audits its rules text, game data, and assets.

## Open5e

Arcane Ledger may use Open5e data through the Open5e API and related imported records.

- Open5e: https://open5e.com/
- Open5e legal information: https://open5e.com/legal
- Open5e API documentation: https://open5e.com/api-docs

Open5e records include source document metadata. Arcane Ledger should preserve that metadata, including license URLs where available, whenever Open5e-backed records are displayed, exported, redistributed, or otherwise reused.
